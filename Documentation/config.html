О config.json

Этот файл содержит в себе конфигурацию Udp и Tcp серверов. При вызове методов create() и recreate() файл config.json открывается каждый раз.

Стандарт файла

{
"address": "localhost", // адрес сервера
"tcpPort": 6240, // порт для Udp сервера
"udpPort": 4062 // порт для Tcp сервера
}
